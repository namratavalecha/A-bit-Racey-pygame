# A-bit-Racey-pygame
A game built using pygame.

## How to build and run the game?

Install cx_Freeze module using pip.

```bash
pip3 install cx_Freeze
```
Edit setup.py file.

Run the following command to build an executable file.

```bash
python3 setup.py build
```
